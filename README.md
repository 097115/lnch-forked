lnch

## About

This is a fork of [`lnch`](https://github.com/oem/lnch) program by [Ömür Özkir](https://github.com/oem/) with the only difference that this repository has some pre-built binaries available. Some information on `lnch` and its usage may be found [here](https://medium.com/njiuko/using-fzf-instead-of-dmenu-2780d184753f).
